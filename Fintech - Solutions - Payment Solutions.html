  • ChequePoint

    DIT / Fintech

    Chequepoint is an MICR cheques capturing system which helps banks to automate their incoming and outgoing cheques processes.

    The system captures the MICR codeline, the cheque's front and back images and cheque encoding. The system also generates inter-bank clearing files which are sent to the clearing house and generates inwards files which are uploaded to the banking systems.

    Chequepoint comes with a range of features including foreign currency cheques processing, stop payments, Image statement, Direct Debits, Electronic Funds Transfer, Signature interfacing, Post Dated Cheques Processing, Proof of Deposit capture, Enhanced security and detailed graphical reporting. These features when combined, makes this new version Full Image Truncation ready allowing the central clearing process to be performed using images without the accompanying physical cheques.

    The system has the following modules and features:
    - Inwards Processing
    - Outwards Processing
    - Payments Processing System (salary, supplier payments processing)
    - Direct Debits System
    - Electronic Funds Transfer
    - Cheque Imaging Module
    - Distributed Cheque Image Viewer (web based)
    - Stop Payments System

    - Multiple-Reader Configuration
    - Archiving System
    - Chequepoint-MIS
    - Chequepoint Bridge (Interface with Core Banking System)
    - Chequepoint Branch Module
    - Comprehensive Security Module
    - Signature Verification System
    - Multi Currency cheque processing

  • ChequePoint (Truncation)

    DIT / Fintech

    Cheque Truncation refers to the process of replacing the physical flow of cheques with electronic information within the 4 legs of Clearing i.e. Outward Clearing, Inward Clearing, Outward Unpaids and Inward Unpaids while retaining the physical cheque within the presenting bank.


    Chequepoint is an MICR cheques capturing system which assists banks automate their inwards and outwards cheques processing operations. The system captures the MICR codeline, the cheques\' front and back images while facilitating cheques encoding and cheques endorsing. It generates inter-banks clearing files with appropriate digital signatures while incorporating the images which are sent to the clearing house. It manages integration with all the major core banking system for both inwards and outwards.

  • Imagemaster

    DIT / Fintech

    The Image Master system provides an imaging solution for the banks' customer specimens such as signatures, photographs and identity cards. It is also a powerful enterprise-wide solution, designed to electronically capture paper-based information and then securely archive, manage and retrieve the documents in electronic format.

    The solution scans the images into the system database, authorizing the images, which are eventually available for view at various points in the bank.

    Image Master comprises of the following modules:
    - Document Imaging - Archive and Retrieval
    - Electronic Document Delivery
    - Business Process Management / Workflow
    - Capture Module: images are scanned into the system database
    - Approval Module: captured information is authorized by second person prior to use
    - Verification Module: Stored images are displayed on-screen for comparison against the customer

    The Verification Module comes into play during:
    - Posting of debit entries into customer account
    - Authorization of posted transactions
    - Verifying against presented cheque's
    - Customer service functions

  • Salary Payment System

    DIT / Fintech

    Fintech has developed a salary payment system "FinSPS" which is a full featured payroll system with full capabilities.

    FinSPS is the most efficient way for companies to pay employees for all types of expenses helping to lower administration costs for funds transferring.

    It features a comprehensive employee master file that stores enough information to handle any payroll processing. A complete history of earnings, benefits, deductions, taxes and hours worked is kept for every employee.

    The system's multiple company payroll processing features can handle the needs of any business, from producing all payroll pay slips in real-time, to asking "what if" questions.

    The system's reporting features provide many standard payroll reports as well as allowing for custom-built reports. Reports can be exported to Ms Word, Ms Excel, rich text and Acrobat PF formats.

  • Web Based Bulk Payments

    DIT / Fintech

    FinPay is a convenient way for customers to make large volume, regular payments with pre-authorised transactions.

    The system is enhanced to transfer fund and payments online with enhanced security. Clients can send instructions for payment to the bank from the comfort of their homes or offices. The system can be used to process RTGS instructions, request for cheques and standing orders, integrate with the clearing system for online requests e.g. EFTs and can be used to pay suppliers and make utility payments.

    For security the system has been coupled with biometric security for authentication of clients and transactions before they are sent to the bank for processing.
